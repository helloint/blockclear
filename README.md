## 红色方块消除蓝色方块游戏说明

### 一、基本规则

- **目标**：通过依次点击四周边缘的红色方块，消灭中间区域的所有蓝色方块。
- **移动方式**：
  - 红色方块只能出现在棋盘四条边的灰色边缘上。
  - 点击某个红色方块后，它会从所在边缘**朝棋盘内部沿直线运动**：
    - 在同一行时，左右直线移动；
    - 在同一列时，上下直线移动。
  - 红色方块以「一格 0.3 秒」的速度逐格向前移动。

### 二、消除判定

- **消除蓝块**：
  - 红色方块在移动途中**第一次碰到蓝色方块时立即停止**；
  - 该红色方块与该蓝色方块**同时消失**。
- **未碰到蓝块**：
  - 如果这条直线方向上没有蓝色方块，红色方块会一直移动到对侧边缘的**前一格**（紧贴灰色边框的最后一个白格）；
  - 到达该格后红色方块直接消失，不会消除任何蓝块。

### 三、胜负条件

- **胜利条件**：
  - 所有蓝色方块都被成功消除，界面会弹出「游戏成功」提示。
- **失败条件**：
  1. 所有红色方块都已经用完，但棋盘上**仍然有蓝色方块**；
  2. 虽然还剩红色方块，但根据当前布局，**已经不存在任何可以消除所有蓝块的点击顺序**（程序会自动判断这一点），会提前弹出「游戏失败（已无消除方案）」。

### 四、提示功能（💡提示）

- 点击控制面板中的 **「💡提示」** 按钮可以开启或关闭提示模式。
- 开启提示后：
  - 程序会根据当前棋盘布局，尝试计算一条可以**完全消除所有蓝块**的红块点击顺序；
  - 若找到一条可行路径，会在对应的红色方块上显示序号：
    - `1` 表示第一步应该点击的红块；
    - `2` 表示第二步；以此类推。
  - 当你按照提示点击某一步后，程序会根据**新的棋盘状态**重新计算后续步骤，并更新剩余红块上的数字。
- 数字含义会随你的实际操作自动调整：
  - 如果你先点了「1 号」，之后的数字会从 2 开始继续；
  - 如果你点了原本显示为「2 号」的红块，只要此时仍有解，重新计算后的第一步会重新编号为 2，保证始终给出一条当前局面的可行方案。

### 五、操作说明

- **行数 / 列数 / 方块数**：
  - 可在页面下方的输入框中调整棋盘的行数、列数以及蓝色 / 红色方块的数量（蓝红数量相同）。
- **新游戏**：
  - 重新随机生成一局新的棋盘布局，包括蓝色方块和红色方块的位置。
- **重新开始**：
  - 使用**同一局的初始布局**重新开始本局游戏（相当于重置到这局开始时的状态）。

### 六、小提示

- 点击红块前可以先打开提示模式，观察提示给出的点击顺序，再决定是否按提示来走。
- 你也可以关掉提示自己思考路线，走到无解时，系统会自动判断并提前给出失败提示。
